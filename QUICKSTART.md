Get Drawdle running in 3 minutes!

## Step 1: Install Dependencies
```bash
npm install
```

## Step 2: Start the Server
```bash
npm start
```

## Step 3: Open Your Browser
Navigate to: **http://localhost:3000**

## First Time User?

1. Click **Register** on the login page
2. Create your account (username, email, password)
3. You'll be automatically logged in
4. Start drawing with today's prompt!

## Already Have an Account?

1. Enter your email and password
2. Click **Login**
3. Continue where you left off

## Basic Usage

### Drawing
- Select brush or eraser tool
- Choose your color
- Adjust brush size
- Draw on the canvas
- Give your art a title
- Click "Save Drawing"

### Gallery
- Click "My Gallery" in the navigation
- View all your saved drawings
- Click any drawing to view full size
- Delete drawings you don't want
